## MisterCocktail

## Getting started

### Requirements

- Ruby 3.1.1 or higher
- DB - `gem install bundler`
- MySQL - `sudo mysql.server start`

### Initial setup

1. Customize the `config/database.yml`
2. Run `rails db:create  db:migrate  db:seed`

### Running MisterCocktail ! 

1. Run `rails s`
